# Transport Demand Prediction 🚍

## Objective
To predict the number of seats that can be sold for each bus ride using machine learning.

## Problem Statement
The goal is to help transport services estimate demand for bus routes traveling to Nairobi.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Steps Performed
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Model Building using Linear Regression
- Model Evaluation

## Key Insights
- Demand varies based on route and time
- Traffic and travel timing impact seat bookings
- Model captures relationship between features and demand

## Conclusion
A Linear Regression model was built to predict transport demand, helping in better planning and resource allocation.
